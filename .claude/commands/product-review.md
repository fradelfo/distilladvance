## Analyse

1. **User value**
   - Problem solved?
   - Significance?
   - Evidence?

2. **Business value**
   - Serves goals?
   - Expected impact?

3. **Scope**
   - MVP-appropriate?
   - What to defer?
   - What's missing?

4. **Risks**
   - Assumptions?
   - What could fail?
   - Dependencies?

5. **Recommendation**
   - Build now / Defer / Research / Don't build
   - If research: what questions?
