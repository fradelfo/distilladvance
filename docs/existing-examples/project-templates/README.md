# Project Templates

Ready-to-use project templates for different types of development work. Each template includes:

- Complete directory structure
- Configured CLAUDE.md file
- Team-optimized settings.json
- Pre-built agent configurations
- Basic automation setup

## Available Templates

### Web Development
- `fullstack-webapp/` - Complete full-stack web application setup
- `frontend-spa/` - Single-page application with modern frontend stack
- `backend-api/` - RESTful API service with database integration

### Data & Analytics
- `data-science/` - Data analysis and ML project setup
- `analytics-dashboard/` - Business intelligence and reporting
- `etl-pipeline/` - Data processing and transformation

### Documentation & Content
- `documentation-site/` - Technical documentation and knowledge base
- `team-handbook/` - Internal team processes and procedures

### Specialized Projects
- `mobile-app/` - Cross-platform mobile development
- `desktop-app/` - Desktop application development
- `devops-infra/` - Infrastructure and deployment automation

## Usage

1. Copy your chosen template to your project directory
2. Run `claude` in the project root
3. Follow the template's specific setup instructions
4. Customize CLAUDE.md for your project needs

Each template is self-contained and includes detailed setup instructions.
